Integrating FlowPaper with DSpace 7.6

This guide will help you integrate FlowPaper (formerly known as FlexPaper) into your DSpace 7.6 installation.

Step 1: Download FlowPaper

Go to the official FlowPaper download page and download the free version.

Step 2: Deploy FlowPaper Files

Extract the downloaded files to a location on your server:

sudo mkdir -p /opt/flowpaper
sudo tar -xzvf path/to/flowpaper_download.tar.gz -C /opt/flowpaper

Move the files to a location accessible by the web server:

sudo cp -r /opt/flowpaper /var/www/html/flowpaper

Step 3: Configure Apache to Serve FlowPaper

Create an Apache configuration file for FlowPaper:

sudo nano /etc/apache2/sites-available/flowpaper.conf

Add the following configuration:

<VirtualHost *:80>
    ServerName flowpaper.yourdomain.com

    DocumentRoot /var/www/html/flowpaper
    <Directory /var/www/html/flowpaper>
        Options Indexes FollowSymLinks
        AllowOverride All
        Require all granted
    </Directory>

    ErrorLog ${APACHE_LOG_DIR}/flowpaper_error.log
    CustomLog ${APACHE_LOG_DIR}/flowpaper_access.log combined
</VirtualHost>

Enable the site and restart Apache:

sudo a2ensite flowpaper.conf
sudo systemctl restart apache2

Step 4: Integrate FlowPaper with DSpace

Modify the DSpace templates to use FlowPaper. Locate the templates in your DSpace installation:

sudo nano /path/to/dspace-xmlui/src/main/webapp/themes/yourtheme/item-view.xsl

Add the following code where you want to embed the FlowPaper viewer:

<iframe src="https://flowpaper.yourdomain.com/flowpaper.js" width="100%" height="600" style="border: none;"></iframe>

Step 5: Test the Configuration

Restart DSpace:

sudo systemctl restart tomcat9

Open a browser and navigate to an item in DSpace that contains a PDF file. The PDF should be rendered using FlowPaper.
