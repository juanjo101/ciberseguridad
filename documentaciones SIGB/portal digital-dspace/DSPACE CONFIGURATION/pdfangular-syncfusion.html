Instructivo para Implementar Syncfusion Angular PDF Viewer en DSpace 7.6

1. Configuración del Entorno

Asegúrate de tener el entorno configurado con Yarn y Angular CLI. Debes estar trabajando dentro del proyecto dspace-angular, que es el frontend de tu instalación de DSpace 7.

2. Instalar Syncfusion PDF Viewer

  1. Navega al directorio de dspace-angular: 
    cd [ruta-de-tu-proyecto]/dspace-angular
  2. Instala el paquete de Syncfusion Angular PDF Viewer utilizando Yarn: 
    yarn add @syncfusion/ej2-angular-pdfviewer

3. Configuración del Módulo Angular

  1. Abre app.module.ts y agrega la importación de PdfViewerModule: 
    
import { NgModule } from '@angular/core';
import { BrowserModule } from '@angular/platform-browser';
import { PdfViewerModule } from '@syncfusion/ej2-angular-pdfviewer';
import { AppComponent } from './app.component';

@NgModule({
  declarations: [
    AppComponent,
    // Otros componentes...
  ],
  imports: [
    BrowserModule,
    PdfViewerModule,  // Agrega este módulo aquí
    // Otros módulos...
  ],
  providers: [],
  bootstrap: [AppComponent]
})
export class AppModule { }

4. Implementación del Visor de PDF en el Componente

  1. Crea un componente nuevo o selecciona el componente donde deseas agregar el visor de PDF. Por ejemplo, puedes crear un nuevo componente llamado pdf-viewer: 
    ng generate component pdf-viewer
  2. Abre el archivo HTML del componente (pdf-viewer.component.html) y añade el siguiente código: 
    
<ejs-pdfviewer id="pdfViewer" [serviceUrl]="'https://ej2services.syncfusion.com/production/web-services/api/pdfviewer'" style="height:600px">
</ejs-pdfviewer>
  3. Configura las opciones del visor para deshabilitar la descarga e impresión en el archivo TypeScript del componente (pdf-viewer.component.ts): 
    
import { Component } from '@angular/core';

@Component({
  selector: 'app-pdf-viewer',
  templateUrl: './pdf-viewer.component.html',
  styleUrls: ['./pdf-viewer.component.css']
})
export class PdfViewerComponent {
  public toolbarSettings: object = {
    showDownload: false, // Deshabilita la opción de descarga
    showPrint: false     // Deshabilita la opción de impresión
  };
}
  4. Modifica el HTML del componente para usar estas configuraciones: 
    
<ejs-pdfviewer id="pdfViewer" [serviceUrl]="'https://ej2services.syncfusion.com/production/web-services/api/pdfviewer'" [toolbarSettings]="toolbarSettings" style="height:600px">
</ejs-pdfviewer>

5. Integración en DSpace

  1. Si deseas que este visor esté disponible en una página específica de tu frontend de DSpace, asegúrate de incluir el nuevo componente en la plantilla correspondiente o enlázalo desde donde necesites visualizar los documentos PDF.
  2. Puedes personalizar más las opciones del visor según las necesidades de tu proyecto, consultando la documentación oficial de Syncfusion.

6. Prueba y Ajustes

  1. Ejecuta el frontend para verificar que el visor de PDF funcione correctamente: 
    yarn start
  2. Abre el navegador y navega a la ruta del componente para confirmar que el visor de PDF se muestra correctamente sin las opciones de impresión y descarga.

Notas Adicionales

  • Backend de Syncfusion: El serviceUrl usado en este ejemplo apunta a un servicio de prueba proporcionado por Syncfusion. Para un entorno de producción, considera configurar tu propio servicio para gestionar los documentos de manera más segura.
  • Licenciamiento: Verifica los términos de la licencia de Syncfusion para asegurar que cumples con las condiciones de uso gratuito o considera opciones de licencia pagada si el proyecto lo requiere.
